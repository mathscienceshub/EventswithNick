SUPABASE CONFIGURATION SNIPPET

Open: assets/js/backend-config.js

Change this:
window.EWN_BACKEND = {
  provider: 'local',
  supabaseUrl: '',
  supabaseAnonKey: ''
};

To this:
window.EWN_BACKEND = {
  provider: 'supabase',
  supabaseUrl: 'https://your-project-id.supabase.co',
  supabaseAnonKey: 'your-anon-public-key-here'
};

Do not use the service_role key in public website files.
